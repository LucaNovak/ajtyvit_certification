## 2. Women Machine Learning and AI Academy
An advanced follow-up program focused on the development, training, and deployment of Artificial Intelligence and Machine Learning models.

* **Data Preparation for ML:** Data collection, Feature Engineering, categorical data conversion, scaling, and train/test split.
* **Predictive Modeling (Supervised Learning):** Evaluating classification and regression tasks using algorithms:
  * Linear and Logistic Regression
  * Decision Trees
  * Random Forest
* **Clustering (Unsupervised Learning):** Data and customer segmentation using the **K-Means** algorithm.
* **Professional Techniques:** Building automated data workflows (**Pipelines**), hyperparameter optimization (Model Tuning), and validation using **Cross-validation**.
* **Final Certification in the Repository:**
  * [Certification_ML_Novakova.ipynb](Certification_ML_Novakova.ipynb)

**Technologies used:** `Python` (libraries: `Scikit-Learn`, `XGBoost`), `ML Pipelines`, `Jupyter Notebook`
